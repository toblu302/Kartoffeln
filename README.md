## Kartoffeln
A UCI-compatible Chess Engine written in C++

### About
Uses 64-bit bitboards to represent the current state of the game.

#### Search
* Negamax with alpha-beta cutoff

#### Evaluation
* Material counting
* PSQT

### Custom UCI commands
perft [depth], i.e "perft 4".
